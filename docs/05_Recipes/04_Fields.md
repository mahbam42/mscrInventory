# Recipe Fields

| **Field** | **Description** |
|-----------|-----------------|
| **Name** | Name of the recipe, usually matching the menu item |
| **Base Size** | Defines the reference size (e.g., 12oz) for scaling |
| **Category** | Groups the recipe (Drinks, Bakery, etc.) |
| **Ingredients** | List of ingredients with quantities and units |
| **Modifiers** | Optional adjustments (milk swaps, syrups, iced options) |
| **Active** | Determines whether the recipe is currently in use |

These fields collectively determine how products are constructed from recipes.
