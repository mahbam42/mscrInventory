# Creating or Editing Recipes

## Creating a New Recipe
1. Go to **Recipes → Manage Recipes**.  
2. Click **Add Recipe**.  
3. Enter the recipe **name**, **category**, and **base size** (typically Small / 12oz).  
4. Add ingredients by selecting them from the dropdown and entering quantities.  
5. Save the recipe.

## Editing an Existing Recipe
1. Navigate to **Manage Recipes**.  
2. Find the recipe and click **Edit (✎)**.  
3. Adjust ingredient amounts or add/remove components.  
4. Save changes.

## Duplicating a Recipe
1. Open the recipe you want to base a new one on.  
2. Select **Duplicate Recipe**.  
3. Rename the new version and adjust quantities or modifiers.  
4. Save it as a new recipe.

**NOTE:** Editing a recipe immediately updates all linked products.
